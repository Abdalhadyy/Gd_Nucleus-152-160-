# IBM-1 Nuclear Structure Simulation — ¹⁵²⁻¹⁶⁰Gd

![HTML](https://img.shields.io/badge/HTML-single%20file-orange)
![Physics](https://img.shields.io/badge/Physics-IBM--1-blue)
![License](https://img.shields.io/badge/license-MIT-green)

Interactive visualization of nuclear shape evolution in even–even **Gadolinium isotopes (¹⁵²–¹⁶⁰Gd)** based on the **Interacting Boson Model (IBM-1)**.

The simulation illustrates how nuclear structure evolves as neutrons are added, transitioning between spherical and deformed shapes.

🔗 **[View Live Simulation](https://abdalhadyy.github.io/Gd_Nucleus-152-160-/Gd_Isotopes/Gd-Isotopes-152-160.html)**

---

## Features

- **3D Nuclear Shape Visualization**  
  Observe the transition: spherical → transitional → deformed.

- **IBM Symmetry Triangle**  
  Locate each isotope within the **U(5), X(5), and SU(3)** symmetry limits.

- **Energy Level Comparison**  
  Compare **experimental energy spectra** with IBM theoretical predictions.

- **Interactive Controls**  
  Adjust neutron number **N = 88 → 96** using a slider or dropdown selector.

- **Evolution Animation**  
  Press **▶ Evolution** to automatically animate the shape transition across all isotopes.

- **Export Figure**  
  Download the current energy level diagram as a PNG for academic use.

## Key Physical Result

The isotope **¹⁵⁴Gd (N = 90)** appears near the **X(5) critical-point symmetry**, representing the transition between spherical U(5) and axially deformed SU(3) nuclear shapes.

| Isotope | N  | Symmetry       | R₄/₂ | P-Factor |
|---------|----|----------------|------|----------|
| ¹⁵²Gd   | 88 | U(5) Spherical | 2.19 | 2.857    |
| ¹⁵⁴Gd   | 90 | **X(5) ★**     | 3.01 | 4.632    |
| ¹⁵⁶Gd   | 92 | SU(3) Deformed | 3.23 | 5.455    |
| ¹⁵⁸Gd   | 94 | SU(3) Deformed | 3.28 | 6.240    |
| ¹⁶⁰Gd   | 96 | SU(3) Deformed | 3.30 | 7.000    |
